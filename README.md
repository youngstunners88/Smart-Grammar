# Smart Grammar

A fully offline lesson library for teachers, covering the Grammar 2, Grammar 3, and Grammar 4 e-books.

## How to open it (no install needed)

1. Unzip this folder anywhere on your computer (Windows, Mac, or Linux).
2. Double-click `index.html`. It opens in your default web browser.
3. Pick **Grammar 2**, **Grammar 3**, or **Grammar 4**, then click any lesson to view it full-screen.
4. Use the left/right arrows (or arrow keys) to move between lessons, and Esc or the × to close.

No internet connection is required once you've downloaded it — everything (images included) is stored locally in this folder.
