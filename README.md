# Smart Grammar

A lesson library for teachers, covering the Grammar 2 and Grammar 3 e-books, viewable online or downloaded for fully offline use.

## Online version

- Picker page (choose a book): https://youngstunners88.github.io/Smart-Grammar/
- Grammar 2 direct: https://youngstunners88.github.io/Grammar-2/
- Grammar 3 direct: https://youngstunners88.github.io/Grammar-3/
- Grammar 4: https://youngstunners88.github.io/Grammar-4/ (not yet included in Smart Grammar — still being finalized)

## Offline downloads

Each book is packaged as its own zip (kept under Base44's file-serving size limit so downloads don't get corrupted):

- Grammar 2 (24MB): https://base44.app/api/apps/6a45db6a5e1aa761e75f1c16/files/mp/public/6a45db6a5e1aa761e75f1c16/7c3784cd1_Grammar2.zip
- Grammar 3 (13MB): https://base44.app/api/apps/6a45db6a5e1aa761e75f1c16/files/mp/public/6a45db6a5e1aa761e75f1c16/9b9240752_Grammar3.zip

To use: unzip anywhere on Windows, Mac, or Linux, then double-click `index.html`. No internet connection or install needed — everything (images included) is stored locally in the folder. Use the left/right arrows (or arrow keys) to move between lessons, Esc or × to close.

## Notes on content

- Grammar 2 is complete: 32 lessons (with b-variants) plus Hello A/B/C intros, 69 pages total, in natural numeric order (Lesson 2 before Lesson 10, etc.).
- Grammar 3 has 76 pages (A through 36b). Still missing two: Lesson 17 (non-b) and Lesson 19b — not yet in the source folder.
- Grammar 4 currently has 19 pages, still being finalized — excluded from this delivery until confirmed complete.
